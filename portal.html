Real-Time Rendering Portal
Last changed: August 12, 2024

This page is devoted to sites and tools we use on a continuing basis. They're personal picks, and reflect our own biases.

  1. Ke-Sen Huang's conference pages has links for papers from all the major computer graphics conferences and workshops. The pages by Tim Rowley are not available directly, but this archive contains them.
  2. Advances in Real-Time Rendering in 3D Graphics and Games, Introduction to Real-Time Ray Tracing, Open Problems in Real-Time Rendering, An Overview of Next-Generation Graphics APIs, and Stylized Rendering in Games SIGGRAPH course materials are hosted on our site.
  3. SIGGRAPH 2023 links, compiled by Stephen Hill. Also see link pages for SIGGRAPH 2023 links SIGGRAPH 2022 links SIGGRAPH 2021 links SIGGRAPH 2020 links SIGGRAPH 2019 links (and SIGGRAPH 2019 ray tracing links), SIGGRAPH 2018, SIGGRAPH 2017, SIGGRAPH 2016, SIGGRAPH 2015, SIGGRAPH 2014, SIGGRAPH 2013, SIGGRAPH 2012 and SIGGRAPH 2011.
  4. YouTube research conference resources: SIGGRAPH and SIGGRAPH Asia, I3D, HPG, EGSR 2024.
  5. Game Developers Conference: see the GDC Vault, and the YouTube channel of talks from 2022 to the present. 2019 links, also 2018, 2017, 2016. There's none we can find for 2015, but before then Javier "Jare" Arevalo collected GDC 2014, 2013, and 2012 presentations.
  6. Graphics Programming weekly - Jendrik Illner summarizes graphics blog articles. Think of it as your one-stop blog. He also has a nice searchable collection.
  7. NVIDIA and AMD (plus GPUOpen) graphics developer sites - demos, code samples, white papers, etc. Other worthwhile code samples at Humus-3D.
  8. Min Chen's list of Computer Graphics Forum State-of-the-Art (STAR), survey, and review papers since 2010.
  9. The Journal of Computer Graphics Techniques - open access (free to all) and many articles include code samples.
  10. Journal of Graphics Tools (JGT) code repository.
  11. Graphics Gems Repository - contains the source code for many graphics algorithms. Search the contents by category, by author, or by book.
  12. Developer sites and mailing lists: GameDev.net is active, as is OpenGL.org, Ogre Forums, GD Algorithms archives dying out but searchable (subscribe), and FlipCode (old, closed, but some good things in the archives).
  13. Game company publication pages: alphabetically, and a few quite dated, but here goes. Frostbite, Guerrilla Games, Unreal Engine, Unity, Ready At Dawn, Tri-Ace, and Activision (and dated Activision).
  14. Commercial research lab pages: NVIDIA Research, Microsoft Research U.S., and Cesium (GIS).
  15. Film company publication pages: Disney (Hyperion renderer specific) and Pixar.
  16. Catlike Coding has tutorials for programming in Unity and Godot.
  17. Acerola gives a bunch of tutorials, specializing in post-process effects. His What is a Graphics Programmer? video is an interesting journey.
  18. Level 80 has a constant stream of information for game artists and content creators.
  19. Game Developer programming page - the successor to Gamasutra's programming page.
  20. Free (and good) books online
  21. Models: McGuire CG Archive, Stanford 3D Scanning, Large Geometric Models (Georgia Tech), Utah 3D Animation, ORCA, Khronos Group glTF, Smithsonian, Disney, Intel, Apple AR Quick Look, Bitterli, pbrt, Mitsuba, TurboSquid, Hair from Cem Yuksel, Free3D, Poly Haven, CGTrader, Sketchfab, Three D Scans, PoliigonBEEPLE (C4D project files), Standard Procedural Databases. USD: Moana, ALab, Scan the World, and ALab 2, Sponza, DPEL from ASWF,and another list.
  22. APIs:
    • OpenGL: SDK, specifications, tutorials, Raster Tek tutorials, extensions and more, GLEW, Mesa, NVIDIA's site.
    • Vulkan: Official site, tutorial, NVIDIA's site.
    • DirectX: SDK, tutorials, Raster Tek tutorials, debugging, version stats, NVIDIA's site.
    • WebGL: 2.0 Specification, browser support, device/browser test, current browser details, debugging/profiling, x3dom, and WebGL Resources page.
    • Metal: developer site.
  23. Paper Search:
    • Google Scholar
    • ACM Digital Library
    • The Journal of Computer Graphics Techniques
    • IEEE Computer Graphics & Applications
    • Eurographics
    • Computer Graphics Forum
  24. Education:
    • Interactive 3D Graphics is an interactive course (MOOC) by Eric using videos (with transcripts) alongside exercises and demos (using three.js) that run in your browser. Focus is on general principles, though specifics, such as shader programming, are also covered.
    • The Rendering Course from TU Wien is a free master's level video course on rendering. More information here.
    • Shader School is highly recommended for learning GLSL; it has a nice way of teaching you step by step. Answers here, as (on the Mac, at least) the one flaw is that compiler errors are not shown.
    • three.js is a scene graph layer and much more atop WebGL. It makes adding 3D web content pretty easy - enjoy the demos on that page, or sample the examples (and even better, commented examples here).
    • Scratchapixel has a series of articles on the basics of computer graphics theory.
    • Shadertoy is unlikely to help you write shaders - I just wanted to sneak it in. Rather, enjoy the extreme shaders (using procedural geometry, ray casting, etc.) that others have written, right in your browser. New shaders every day.
  25. The Level Up Report by Mark DeLoura. Stopped publishing in 2018, but was a free weekly that provided pointers to all sorts of developments and resources for learning through games, coding, and making.
  26. Game Developer Magazine - stopped printing in 2013 (which sounds ancient, but there are some good articles on the basics). All issues are available free for download.
  27. Graphics and Game Gems Database is dated (2016), but lets you search the various popular interactive graphics series, browse each book's article titles, and get bibtex entries. AI Wisdom is a simila sites. They leave out a few newer (and older) books, but have the advantage of being able to browse by category.
  28. ACM TOG's Software Tools and Resources page is ancient (2015), but has much about both graphics software and where to get BRDF data and other resources.
  29. Gameconfs - game-related conferences.
  30. Geometric Tools - there are many different code snippets and tutorials here to do all sorts of graphics operations, with a focus on computational geometry and intersection methods.
  31. 3D Object Intersection Page - where to find articles and code on this topic.
  32. Virtual Terrain Project - an old (2013) repository of algorithms about and models of terrain, vegetation, natural phenomena, etc.
  33. comp.graphics.algorithms FAQ - truly ancient (last updated 2003), but still full of computational goodness.
  34. Eric Weisstein's World of Mathematics is an incredible (though often dense) resource for mathematical definitions.
  35. Steam's hardware survey tracks what is currently used by their subscribers.
  36. Dead End Thrills makes some excellent renders of current games.
  37. Tech Power Up has an up-to-date summary of the clock speed, memory size, and other characteristics for every major consumer PC GPU. Tech ARP has similar GPU charts for workstations, PCs, and mobile devices.
  38. GPU Review Sites - these sites often have in-depth analysis of GPU features:
    • Tom's Hardware Guide

    • Björn's 3D World
    • [H]ardForum
  39. Game Related - Metacritic for meta-ratings, VG Chartz for console and handheld sales figures, gamedevmap for developer locations.
  40. Github and SourceForge are popular places to browse for free, open-source software applications.
  41. Blogmania: Your first (and possibly only) stop could be Jendrik Illner's, which summarizes all sorts of graphics-related posts. Also, most bloggers (and non-bloggers) also have a Twitter feed - find and follow them.   42. Real Artifacts - a collection of real-world rendering bugs, hosted here.
Special bonus site: you need to visit it only once, but Maxima is worth listing here. It is a free version of Macsyma (which is similar to Mathematica and Maple). If you work with equations and do not have $1500 to spare, you need this.

Resources Hosted Onsite

Beyond the books list and object/object intersections page, the major resources hosted here are:

  • Ke-Sen Huang's conference pages - links to conference articles and related materials.
  • Advances in Real-Time Rendering in 3D Graphics and Games - the course materials from SIGGRAPH 2010 on.
  • Open Problems in Real-Time Rendering - the course materials from SIGGRAPH 2015 on.
  • An Overview of Next-Generation Graphics APIs - the course materials from SIGGRAPH 2015.
  • Stylized Rendering in Games - the course materials from SIGGRAPH 2010.
  • Unity: iOS and Android - Cross-Platform Challenges and Solutions - at SIGGRAPH 2012 Renaldas Zioma presented this talk about mobile development. I liked it so much I asked to host it.
  • Fantasy Graphics League - an ancient (2000-2005) yearly series of contests at picking the best research lab. All Eric can say about this is that Ren Ng told him, "I once saw an unnamed professor yell at a student to finish an argument, 'Oh yeah? Well come to talk to me when you're worth 400 Quatloos on Fantasy Graphics League.'"
  • Globillum mailing list archives - the archive of the global illumination mailing list, which had discussions about all sorts of topics in its day.

Computer Programs and Services

 Here are some computer programs and services we like, in rough order of awesomeness.
  • Dropbox - you should know this already, but just in case: creates a special folder that is shared among three of your machines for free. You can also share with others privately or publicly. There are competitors, such as OneDrive.
  • iPad: our only recommendation is GoodReader - lets you stuff and view documents onto your iPad, syncs nicely with Dropbox.
  • Ninite - installing stuff on a new PC? Save yourself a few hours by just checking off the things you want and getting one program to install them all, without prompts. Also a nice site for finding programs to try out.
  • WizTree and WinDirStat (Disk Inventory X on the Mac) - shows where all your disk space has gone in an excellent visual fashion. WizTree is new and much faster than WinDirStat. Try clicking on everything you see in the display to get more info, delete, etc. Free.
  • Beyond Compare - an excellent file comparison program. It even compares images, also showing their alpha layer. Shareware, but the trial gives 30 days of use. Our review here.
  • LICEcap - icky name, nice utility for creating animated GIFs of what's happening on the screen. Use this for capturing program behavior, to show what problem a bug is causing.
  • cppcheck - easy to use and free, it will probably find some coding errors that your compiler didn't notice. More about it here.
  • Visual Studio Code - the most confusing name for a product ever, it has nothing to do with the Visual Studio IDE. A nice, free text editor, good for JavaScript development in particular.
  • Assimp is an open source library that reads a wide variety of 3D model file formats (and writes out a few). It also can perform some mesh clean-up functions.
  • Simple 3D model reader code: tiny OBJ loader and tiny glTF loader.
  • MeshLab is a free package that's good for mesh manipulation and translation. It's got rough edges all over - expect crashes and lockups on advanced functionality - but it's got a huge range of functions.
  • 3D Viewers: there are lots. Meshlab, mentioned above, does not handle transparency well. We particularly like G3D Viewer (full website) and FBX Review.
  • Screen capture: we've looked far and wide. FastStone Capture is shareware that costs $20, but does a lot, including annotation. Greenshot is a favorite free one.
  • Image viewers: it's a matter of taste and needs. IrfanView, XnView, and FastStone Image Viewer are popular generic viewers. ddsview lets you look at mipmap levels (for DDS) and examine colors and alpha easily (plus source code is available).
  • Video editing: OBS Studio is reasonable; you can even use it to run conferences. The free version of DaVinci Resolve is supposed to be quite good.
  • Everything finds files and folders instantly.
  • Glary Utilities, free for home use, has worthwhile cleanup tools and other utilities.
  • Shader-related: NShader adds syntax highlighting for shaders developed in Visual Studio, or you can get most of the way there in MSVC by "Options | Text Editor | File Extension" and setting extension .fx (.fxh, etc.), choosing Microsoft Visual C++, then clicking "Add". Other shader-related tools are described here.
  • Debugging: start with RenderDoc if you need a graphics debugger.
  • Performance: there are many tools in this area, here's an (old) start.
